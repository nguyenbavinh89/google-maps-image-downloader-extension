# Google Maps Image Downloader Extension

Official browser extension for downloadmapimage.com.

## Supported Browsers

- Google Chrome
- Microsoft Edge
- Mozilla Firefox
- Opera

## Features

- Adds a "Get Image" button to Google Maps.
- Download photos from Google Maps.
- Download Street View images.
- One-click integration with downloadmapimage.com.

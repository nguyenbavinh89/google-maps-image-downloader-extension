# Google Maps Image Downloader Extension

Official browser extension for DownloadMapImage.com.

## Overview

Google Maps Image Downloader Extension helps users download Google Maps photos and Street View images directly from the Google Maps interface.

The extension integrates a convenient "Get Image" button into Google Maps, allowing users to access available photos quickly and efficiently.

## Features

- Download Google Maps place photos
- Download Street View images
- Support Chrome
- Support Edge
- Support Firefox
- Support Opera
